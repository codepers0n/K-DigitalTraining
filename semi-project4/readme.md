## 해리포터 등장인물 데이터에 대한 텍스트 유사도 분석

데이터 수집 : 나무위키 문서의 기숙사별 텍스트를 크롤링으로 수집(기숙사별 정보를 수집하고 싶었으나 국내외 영화 리뷰, 위키피디아 등에 기숙사별로 정리 된 데이터를 찾는 것이 쉽지 않아 나무위키 문서를 활용)
수집한 데이터를 분석 해 유저의 입력 값과 유사도 분석을 통해 유저의 삼총사를 찾아준다.
